# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript with turn-based logic, win detection, and draw handling.

## 🚀 Features
- Two-player gameplay
- Win & draw detection
- Reset functionality
- Clean modern UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
